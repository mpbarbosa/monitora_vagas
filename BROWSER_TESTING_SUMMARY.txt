QUICK START GUIDE
================================================================================

PREREQUISITES:
--------------
1. Python 3.11+ (already installed)
2. Selenium 4.39.0 (already installed via requirements.txt)
3. Chrome/Chromium browser
   
   Install if needed:
   $ sudo apt-get install chromium-browser

RUNNING SELENIUM TESTS (RECOMMENDED):
--------------------------------------
# Terminal 1: Start development server
$ npm start

# Terminal 2: Run Selenium tests
$ npm run test:browser:selenium

# Or run against production
$ npm run test:browser:selenium:prod

OPTIONAL - INSTALL PLAYWRIGHT:
-------------------------------
$ pip install playwright==1.40.0
$ python -m playwright install chromium

# Run Playwright tests
$ npm run test:browser:playwright

# Run all browser tests
$ npm run test:browser:all

TROUBLESHOOTING
================================================================================

ISSUE: Chrome not found
SOLUTION: sudo apt-get install chromium-browser

ISSUE: Tests timeout
SOLUTION: 
  1. Check app is running: npm start
  2. Verify network: curl http://localhost:8080
  3. Increase timeout in test file

ISSUE: Selenium not working
SOLUTION: pip install --upgrade selenium

ISSUE: Playwright not installed
SOLUTION:
  pip install playwright==1.40.0
  python -m playwright install chromium

For more help, see: docs/BROWSER_TESTING_GUIDE.md
